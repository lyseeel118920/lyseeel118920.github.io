1.快速排序

快速排序的基本流程

1.选择分界点/枢轴量x，通常x取q[l];

2.调整区间：调整后左边区间都小于等于x，右边区间都大于等于x;

3.递归处理左右两区间.

快速排序的代码模板

值得注意的是，博客中所有的代码示例都是验证过的正确的C++代码，可以直接运行或调用。 
#include<iostream>
using namespace std;
void quick_sort(int q[],int l,int r)
{
   if(l>=r) return ;//结束条件
	int x=q[l+r>>1]; //step1
	int i=l-1,j=r+1;
	
	while(j>i)//step2
	{
		do i++;while(q[i]<x);
		do j--;while(q[j]>x);
     	if(i<j)swap(q[i],q[j]);
	}
	quick_sort(q,l,j);//step3
     	quick_sort(q,j+1,r);
}
int main()
{
	int i,n,q[100000];
	scanf("%d",&n);
	for(i=0;i<n;i++) scanf("%d",&q[i]);
	quick_sort(q,0,n-1);
	for(i=0;i<n;i++) printf("%d ",q[i]);
	return 0;
}

快速排序分析

1.算法采用的是分治思想，分而治之;

2.算法的平均时间复杂度为O(nlogn)，最坏时间复杂度为O(n^2);

3.快速排序是不稳定的.

2.归并排序

归并排序的基本流程

1.计算中间点：$mid = \frac{r + l}{2}$

2.给左右两区间递归排序

3.给排好序的两个区间归并成一个有序区间

归并排序的代码模板

#include<iostream>
using namespace std;

	int temp[1000];
void merge_soft(int* p,int l,int r)
{
	if(l>=r) return;
	int mid=l+r>>1;
	merge_soft(p,l,mid);
	merge_soft(p,mid+1,r);
	int i=l,j=mid+1,k=l;
	while(i<=mid&&j<=r)
	{
	 if(p[i]<=p[j]) temp[k++]=p[i++];
	 else           temp[k++]=p[j++];
	}
	while(i<=mid) temp[k++]=p[i++];
	while(j<=r)   temp[k++]=p[j++];
	for(i=l;i<=r;i++) p[i]=temp[i];
}
int main()
{
	int n;
	scanf("%d\n",&n);
	int p[1000],i;
	for(i=0;i<n;i++)scanf("%d",&p[i]);
	merge_soft(p,0,n-1);
	for(i=0;i<n;i++)printf("%d ",p[i]);
	return 0;
}

归并排序分析

1.算法也采用的是分治思想;

2.归并排序的时间复杂度为O(nlogn)；

3.归并排序是稳定的.

3.二分查找

整数二分的基本流程

二分查找法步骤

1.确定目标所满足的性质，是求左边界还是右边界（>= 还是 <=）

2.确定 mid 的计算方法：

  • A. 对于左边界问题，$mid = \frac{l + r}{2} $
  • B. 对于右边界问题，$mid = \frac{l + r + 1}{2} $

3.更新目标区间范围：

  • A. 对于左边界问题，如果 check(mid) = true，则使 r = mid；如果 check(mid) = false，则使 l = mid + 1
  • B. 对于右边界问题，如果 check(mid) = true，则使 l = mid；如果 check(mid) = false，则使 r = mid - 1

该问题通过多次迭代，缩小目标所在的区间范围（初始区间范围为 $[l, r]$）。

整数二分的代码模板

模板中的check(mid)函数需要根据查找问题的不同自行设计。 
int bsearch_1(int l,int r) //左边界问题
{  
	while(l<r)
	{
     int mid=l+r>>1;
	 if(check(mid))	r=mid;
	 else	l=mid+1;
	}
	return l;
}

int bsearch_2(int l,int r)//右边界问题
{  
	while(l<r)
	{
     int mid=l+r+1>>1;
	 if(check(mid))	l=mid;
	 else	r=mid-1;
	}
	return l;
}

浮点数二分

浮点数二分不需要考虑边界问题，因此只需要在第一套模板的基础上，修改while条件为r-l>某一个极小精度值，即满足r-l趋于0时，才循环停止。
