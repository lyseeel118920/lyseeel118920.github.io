快速排序的基本流程

1.选择分界点/枢轴量x，通常x取q[l];

2.调整区间：调整后左边区间都小于等于x，右边区间都大于等于x;

3.递归处理左右两区间.

快速排序的代码模板

#include<iostream>
using namespace std;
void quick_sort(int q[],int l,int r)
{
   if(l>=r) return ;//结束条件
	int x=q[l]; //step1
	int i=l-1,j=r+1;
	
	while(j>i)//step2
	{
		do i++;while(q[i]<x);
		do j--;while(q[j]>x);
     	if(i<j)swap(q[l],q[r]);
     	quick_sort(q,l,j);//step3
     	quick_sort(q,j+1,r);
	}
	
}
int main()
{
	int i,n,q[1000];
	scanf("%d",&n);
	for(i=0;i<n;i++) scanf("%d",&q[i]);
	quick_sort(q,0,n-1);
	for(i=0;i<n;i++) printf("%d ",q[i]);
	return 0;
}

快速排序分析

1.算法采用的是分治思想，分而治之;

2.算法的平均时间复杂度为O(nlogn)，最坏时间复杂度为O(n^2);

3.快速排序是不稳定的.
